# Offline fonts

Fonts are loaded from this folder so the app works without internet.

## Required files

Download from [Google Fonts](https://fonts.google.com) and place the TTF files as follows.

### Playfair_Display/
- `PlayfairDisplay-Regular.ttf` (weight 400)
- `PlayfairDisplay-Bold.ttf` (weight 700)
- `PlayfairDisplay-Italic.ttf` (weight 400, italic)

### Poppins/
- `Poppins-Light.ttf` (weight 300)
- `Poppins-Regular.ttf` (weight 400)
- `Poppins-Medium.ttf` (weight 500)
- `Poppins-SemiBold.ttf` (weight 600)
- `Poppins-Bold.ttf` (weight 700)

If your download uses different names (e.g. from the Google Fonts zip), rename them to match the list above. The main CSS that uses these is `public/css/fonts.css`.
